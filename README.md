# OperatorHub.IO

## Welcome to OperatorHub

Operators deliver the automation advantages of cloud services like provisioning, scaling, and backup/restore while being
able to run anywhere that Kubernetes can run.

### Build

##### Frontend

```
$ cd Frontend
$ yarn
$ yarn build

#### Node Server
cd server
yarn
```

### Run

##### Production Mode

```
$ cd server
$ yarn server

http://0.0.0.0:9065/
```

##### Development Mode

```
$cd frontend
yarn start
http://0.0.0.0:9060/
```

##### Mock Mode

```
$cd frontend
yarn mock
```
